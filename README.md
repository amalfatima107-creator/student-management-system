# student-management-system
A C++ program that manages coursework and research students using inheritance and polymorphism.
Student Management System (C++)


## Features
- Base `Student` class with derived `Student_Course` and `Student_Research` classes
- Polymorphic `reportGrade()` method
- CSV file I/O (load and export)
- Menu-driven interface:
  - Remove student by ID
  - Output all students
  - Calculate above/below average marks
  - Report grade for a student
  - Sort by ID (insertion sort)
  - Export sorted list to CSV

## OOP Concepts
- Inheritance
- Polymorphism
- Encapsulation
- Code reuse

## How to Run (Eclipse)
1. Import the project into Eclipse IDE
2. Build and run `main.cpp`
